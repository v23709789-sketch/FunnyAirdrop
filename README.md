📋 FunnyAirdrop - Плагин на аирдропы​

🎯 Основной функционал​
✨ Создание и управление аирдропами с настраиваемым лутом
📦 Случайная генерация предметов с шансами выпадения
🎨 RGB цвета в названиях и сообщениях
🔧 Интеграция с WorldEdit для установки схематик
🛡️ Интеграция с WorldGuard для защиты регионов
📊 Система BossBar для отображения статуса
💬 Голограммы над аирдропами (DecentHolograms)
⏱️ Настраиваемые таймеры открытия и закрытия

🎪 Система событий​
📡 Каждый аирдроп поддерживает настраиваемые события с командами:
• START - запуск аирдропа
• END - завершение аирдропа
• CLICK_CLOSED - клик по закрытому
• CLICK_OPENED - клик по открытому
• CLICK_LOOTED - полное разграбление
• TIMER - таймер обратного отсчёта
• PICKUP_DELAY - задержка поднятия предметов
• LOOT_DISGUISE - маскировка лута
• DROP_LOOT - выбрасывание на землю
• BOSS_BAR_CLOSED - босс-бар закрытого
• BOSS_BAR_OPENED - босс-бар открытого
• BOSS_BAR_CLICK_ACTIVATE - босс-бар ожидания активации

⚙️ Гибкая настройка аирдропов​
📁 airdrops/*.yml - конфигурации аирдропов
📁 listener.yml - настройки обработчиков событий
📁 schematics/ - папка со схематиками
📁 config.yml - основные настройки плагина

🛡️ Защита и безопасность​
✅ Автоматический откат схематик через WorldEdit
✅ Защита регионов от взрывов и построек
✅ Настраиваемые флаги WorldGuard
✅ Автоматическое закрытие инвентарей
✅ Безопасная работа с чанками

🎯 Placeholder​
%funnyairdrop_<имя>_<плейсхолдер>%

📊 Статус:
🔹 active - активен (true/false)
🔹 opened - открыт (true/false)
🔹 looted - разграблен (true/false)
🔹 status - Открыт/Закрыт/Ожидает спавна/Не активен
🔹 auto_respawn - авто-респавн (true/false)
🔹 click_to_activate - активация по клику (true/false)
🔹 static_mode - статичный режим (true/false)

⏱️ Время:
🔹 time - оставшееся время (сек)
🔹 time_formatted - оставшееся время (X мин. Y сек.)
🔹 time_to_open - время до открытия (сек)
🔹 time_to_open_formatted - время до открытия (X мин. Y сек.)
🔹 time_to_respawn - время до респавна (сек)
🔹 time_to_respawn_formatted - время до респавна (X мин. Y сек.)

📊 Статичные значения:
🔹 open_time - время открытия (сек)
🔹 close_time - общее время ивента (сек)
🔹 auto_respawn_time - интервал респавна (сек)
🔹 spawn_time - время спавна (HH:mm:ss)

📝 Информация:
🔹 displayname - отображаемое имя
🔹 name - системное имя
🔹 world - мир
🔹 radius - радиус спавна
🔹 inventory_size - размер инвентаря
🔹 hologram_height - высота голограммы
🔹 world_guard_radius - радиус WG
🔹 min_online - мин. онлайн

📍 Координаты:
🔹 x - X координата
🔹 y - Y координата
🔹 z - Z координата

👥 Онлайн:
🔹 online - текущий онлайн сервера

📦 Зависимости​
🔷 WorldEdit - для работы со схематиками
🔷 WorldGuard - для защиты регионов
🔷 DecentHolograms - для голограмм (опционально)
